# Gado 11 - Portal do Confinador

Este projeto é uma implementação da interface de login do Portal do Confinador da Gado 11, desenvolvido com Next.js 14, TypeScript e Tailwind CSS.

## Tecnologias Utilizadas

- **Framework:** Next.js 14 (App Router)
- **Linguagem:** TypeScript
- **Estilização:** Tailwind CSS (v4)
- **Ícones:** Lucide React
- **Fontes:** Manrope (via next/font)

## Funcionalidades

- Layout responsivo (Mobile-first) com design split-screen em desktop.
- Tema escuro personalizado com cores da marca (#13ec5b).
- Componentes reutilizáveis (Button, Input, StatCard).
- Validação visual de campos e estados de foco.
- Otimização de imagens com `next/image`.
- Simulação de login com feedback de carregamento.

## Como Executar

1. Instale as dependências:

```bash
npm install
```

2. Execute o servidor de desenvolvimento:

```bash
npm run dev
```

3. Acesse `http://localhost:3000`.
